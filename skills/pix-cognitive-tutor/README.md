## 🎯 项目概述

认知导师技能是一个教育代理，旨在通过结构化的五步学习方法帮助学习者理解新概念和技术。它遵循问题基于学习和认知学徒制的原则，创建完整的"问题-探索-解答"认知闭环。

## ✨ 核心特性

| 特性 | 描述 |
| :--- | :--- |
| **结构化五步教学法** | 场景 → 局限 → 概念 → 核心操作 → 实现 |
| **基于故事的学习** | 引人入胜的、相关的场景，创建学习背景 |
| **渐进式复杂性** | 从基础开始，然后引入高级概念 |
| **实用示例** | 每个步骤都有具体的代码和实现样本 |
| **清晰解释** | 避免过多的行话和技术术语 |
| **完整学习闭环** | 从问题识别到解决方案实现 |

## 🛠️ 工作原理

认知导师遵循一个经过验证的五步流程来解释任何新概念：

### 1. 提出场景 (Scenario)
构建一个具体的、对初学者友好的场景或问题背景，展示真实世界的应用。

### 2. 发现局限 (Limitation)
展示如果不使用新概念，现有基础方法会遇到的痛点、错误或无法解决的问题。

### 3. 引入概念 (Concept)
引入核心概念并解释它如何解决在局限步骤中识别的问题。

### 4. 核心操作 (Core Mechanics)
拆解概念的基本构成要素、语法或工作原理，专注于基础而不是高级变体。

### 5. 功能实现 (Implementation)
展示如何逐步应用概念来解决第一步中的场景问题，提供可运行的代码或解决方案。
